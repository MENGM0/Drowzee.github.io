win10安装ubuntu18.04双系统

发表于 2019-02-12 | 分类于 ubuntu
字数统计: 294

主要步骤*

Grub界面>

其中 hd0,msdos7为存在/boot目录的磁盘（ls试出来的）

  • ls (hd0,msdos7)/
  • set root=(hd0,msdos7)
  • set prefix=(hd0,msdos7)/boot/grub
  • insmod normal
  • normal

开机

按esc，进入BIOS。进入f10打开传统模式。f9，选中带有efi标识的u盘，

按e，在quiet splash ···后添加acpi_osi=linux nomodeset，f10。（此操作为禁用n卡，分辨率会降低，使用 alt +f7拖动屏幕）

分区>

ext4 /home 70GB

ext4 / 30GB

ext4 /boot 300MB(挂载设备与此区一致)

swap 8GB

恢复分辨率>

点击软件包里的“附件驱动”，切换n卡驱动。

修改启动项>

sudo gedit /boot/grub /grub.cfg

set default=”0” 改为 set default=”2”
